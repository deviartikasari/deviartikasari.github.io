## üöÄ How to Use

### Local Development
1. Simply open `index.html` in any modern web browser
2. No build process or dependencies required
3. Works offline - all assets are self-contained

### Deployment Options

#### Option 1: GitHub Pages
1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Settings ‚Üí Pages
4. Select main branch and root folder
5. Your site will be live at `https://yourusername.github.io/repo-name`

#### Option 2: Netlify (Recommended)
1. Drag and drop the folder to [Netlify Drop](https://app.netlify.com/drop)
2. Get instant deployment with custom domain support
3. Free HTTPS and global CDN

#### Option 3: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts for deployment

#### Option 4: Traditional Web Hosting
1. Upload all files via FTP to your web host
2. Ensure `index.html` is in the root directory
3. Access via your domain name

## üé® Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --color-primary: #FF8B4D;        /* Main accent color */
    --color-primary-dark: #E67633;   /* Darker variant */
    --color-primary-light: #FFA366;  /* Lighter variant */
    --color-primary-bg: #FFF5F0;     /* Background tint */
}
```

### Content
All content is in `index.html`. Search for section IDs:
- `#home` - Hero section
- `#about` - About section
- `#skills` - Skills section
- `#experience` - Experience section
- `#academic` - Academic section
- `#portfolio` - Portfolio section
- `#contact` - Contact section

### Typography
Change the font in `index.html` head section:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@300;400;500;600;700;800&display=swap" rel="stylesheet">
```

Then update CSS:
```css
:root {
    --font-family: 'YourFont', sans-serif;
}
```
